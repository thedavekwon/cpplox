C++ implementation of Lox in [Crafting Interpreters](https://github.com/munificent/craftinginterpreters).

# Building 
```
mkdir build
cd build
cmake ..
make
```

# Testing
```
cd build
ctest
```
# Running Scripts
```
cd build
./cpplox_run <SCRIPT_PATH>
```

# Running REPL
```
cd build
./cpplox_run
```
